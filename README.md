# Yapay Zeka Tabanlı RL ile Tıbbi Doz Ayarlaması
Bu proje, "Memory Test on Drugged Islanders" veri setini kullanarak anti-anksiyete ilaçlarının hafıza performansı üzerindeki etkilerini inceledi. Pekiştirmeli öğrenme (Reinforcement Learning - RL) tabanlı bir model geliştirildi ve offline RL yaklaşımıyla ilaç dozajı kararları optimize edildi. Ajan, mevcut veriler üzerinden öğrenerek ilaç dozunu artırma, sabit tutma veya azaltma kararlarını verdi. Model, PyTorch ve OpenAI Gym kullanılarak simüle edilen özel bir ortamda eğitildi. Eğitim sürecinde model, ödülleri maksimize ederek daha etkili dozaj kararları verdi. Optimizasyon süreci ile modelin doğruluğu artırıldı. Sonuçlar, ilaç dozajının hafıza performansını iyileştirdiğini ve bireysel farklılıkların önemli olduğunu göstermektedir.
